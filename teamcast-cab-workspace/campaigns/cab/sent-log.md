# Sent Log

> Record of all emails sent in the CAB campaign.
> Updated by the Sending Agent (or manually in Phase 1).

---

## Format

| Send ID | Draft ID | Contact ID | First Name | Company | Subject | Template | Mailbox | Sent At | Delivery Status | Bounce Flag |
|---------|----------|-----------|-----------|---------|---------|----------|---------|---------|----------------|-------------|
| [pending] | | | | | | | | | | |

---

## Delivery Status Key
- **delivered**: Email accepted by recipient's mail server
- **bounced_soft**: Temporary delivery failure (retry eligible)
- **bounced_hard**: Permanent delivery failure (suppress contact)
- **unknown**: Status not yet confirmed
