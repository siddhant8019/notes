## Campaign
Teamcast Customer Advisory Board Outreach

## Status
**SETUP** — Building workspace and agent infrastructure

## Objective
Book introductory conversations with senior TA and People leaders at scaling tech companies through CAB invitation framing.

## Key Parameters
| Parameter | Value |
|-----------|-------|
| Campaign type | CAB invitation (soft-entry) |
| Outreach voice | Utkarsh Wagh, Founder |
| Target industries | Tech, SaaS, AI, FinTech, HealthTech, Cloud, Cybersecurity, Enterprise SaaS |
| Target headcount | 0–500 |
| Growth filter | Recently funded OR 15%+ headcount growth |
| Geography | NA, Europe, APAC |
| Primary titles | TA/Recruiting leadership |
| Secondary titles | C-suite, VP HR, People Ops |
| Daily send limit | 20-30 (initial) |
| Email sequence | Email 1/3 (initial) → Email 2 (follow-up) |

## Metric Thresholds
| Metric | Threshold | Action if Breached |
|--------|-----------|-------------------|
| Bounce rate | > 5% per batch | Halt campaign |
| QA pass rate | < 60% per batch | Halt campaign |
| Negative/hostile replies | > 3 per day | Pause and review |
