# Approved Drafts

> Emails that have passed QA and are ready for sending (or human review in Phase 1).
> Updated by the QA Agent.

---

## Format

| Draft ID | Contact ID | First Name | Company | Subject (Primary) | Subject (Alt) | Template | Personalization Score | QA Score | Approved Date | Send Status |
|----------|-----------|-----------|---------|-------------------|---------------|----------|----------------------|----------|--------------|-------------|
| [pending] | | | | | | | | | | |

---

## Send Status Key
- **ready**: Approved and ready to send
- **human_review**: Waiting for human approval (Phase 1)
- **scheduled**: Scheduled for sending
- **sent**: Successfully sent
- **held**: Held back for batch-level reasons (too many from same domain, daily limit reached)

---

## Phase 1 Note
In Phase 1, ALL approved drafts go to `human_review` status. No auto-sending.
The operator reviews and manually approves each draft before it enters the send queue.
